프로젝트 개요
Collaboard

핵심
1. 복잡한 상태 관리 (보드 - 리스트 - 카드 3단구조)
2. 실시간 멀티유저 동기화
3. 높은 성능 (100+ 카드 60fps 유지)
4. 접근성 (키보드 네비게이션)
5. 80%+ 테스트 커버리지

MVP 기능 목록

1. 보드 관리
   - 보드 생성/수정/삭제
   - 보드 목록 조회

2. 리스트 관리
   - 리스트 생성/수정/삭제
   - 리스트 순서 변경 (Drag & Drop)

3. 카드 관리
   - 카드 생성/수정/삭제
   - 카드 이동 (리스트 간)
   - 카드 상세 (제목, 설명, 라벨, 담당자)

4. 실시간 협업
   - 다른 사용자 변경사항 실시간 반영
   - 현재 접속 중인 사용자 표시
   - 충돌 방지 (낙관적 업데이트)

5. 인증
   - 이메일 로그인
   - 게스트 모드 (로컬 스토리지)

2순위 기능 목록
- 카드 코멘트
- 파일 첨부 (S3)
- 활동 로그
- 검색/필터
- 다크모드
